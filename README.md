# Responsive-Design-Framework

## English

### Professional Responsive Design Framework by Gabriel Demetrios Lafis


This repository presents a professional and modern **Responsive Design Framework** developed by Gabriel Demetrios Lafis. It is designed to facilitate the creation of highly responsive and visually appealing web interfaces, ensuring optimal user experience across various devices and screen sizes. The framework emphasizes clean code, modularity, and ease of use, making it an ideal solution for developers seeking a robust foundation for their web projects.

### Features

*   **Responsive Grid System:** A flexible and intuitive grid system that adapts seamlessly to different screen dimensions.
*   **Modern CSS Components:** A collection of professionally designed and reusable UI components, including buttons, forms, and cards.
*   **Utility Classes:** A comprehensive set of utility classes for rapid styling and customization.
*   **Dark Mode Support:** Built-in support for dark mode, enhancing user comfort and aesthetic appeal.
*   **Animations:** Subtle and engaging CSS animations to improve user interaction and visual feedback.
*   **Clean Code:** Adherence to modern CSS best practices and linting for maintainable code.
*   **Easy to Integrate:** Designed for straightforward integration into existing or new web projects.

### Technologies Used

*   HTML5
*   CSS3 (with Custom Properties)
*   Stylelint for CSS linting

### Setup and Usage

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/galafis/Responsive-Design-Framework.git
    ```

2.  **Navigate to the project directory:**

    ```bash
    cd Responsive-Design-Framework
    ```

3.  **Open `src/index.html` in your browser** to see the demo.

    Alternatively, you can serve it with a local server:

    ```bash
    python -m http.server 8000
    # Then visit http://localhost:8000
    ```

## Português

### Framework de Design Responsivo Profissional por Gabriel Demetrios Lafis


Este repositório apresenta um **Framework de Design Responsivo** profissional e moderno desenvolvido por Gabriel Demetrios Lafis. Ele foi projetado para facilitar a criação de interfaces web altamente responsivas e visualmente atraentes, garantindo uma experiência de usuário ideal em vários dispositivos e tamanhos de tela. O framework enfatiza código limpo, modularidade e facilidade de uso, tornando-o uma solução ideal para desenvolvedores que buscam uma base robusta para seus projetos web.

### Funcionalidades

*   **Sistema de Grid Responsivo:** Um sistema de grid flexível e intuitivo que se adapta perfeitamente a diferentes dimensões de tela.
*   **Componentes CSS Modernos:** Uma coleção de componentes de UI profissionalmente projetados e reutilizáveis, incluindo botões, formulários e cartões.
*   **Classes de Utilidade:** Um conjunto abrangente de classes de utilidade para estilização e personalização rápidas.
*   **Suporte a Modo Escuro:** Suporte integrado para modo escuro, aumentando o conforto do usuário e o apelo estético.
*   **Código Limpo:** Adesão às melhores práticas modernas de CSS e linting para código de fácil manutenção.
*   **Fácil Integração:** Projetado para integração direta em projetos web existentes ou novos.

### Tecnologias Utilizadas

*   HTML5
*   CSS3 (com Propriedades Personalizadas)
*   Stylelint para linting de CSS
